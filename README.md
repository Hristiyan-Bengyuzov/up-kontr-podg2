Зад 1:
Напишете функция със сигнатура: void stringManip(char* str)
Функцията трябва да преобразува низа str, като премине последователно през всички символи и извърши следните операции: 
Всички латински букви се преобразуват така: стават поредната цифра от азбуката
 ('A' -> 1, 'a' -> 1, 'z' -> 26) 

Всички цифри '0'…'9' се заместват с числото (цифрата + 3) % 10. 
Например: '7' → '0', '9' → '2', '3' → '6'. 

Символът '_' се заменя с '-'. 

Всички интервали ' ' се премахват от низа

Зад 2:	
Напишете функция със сигнатура: int maxLexDiff(const char* str)
Тя трябва да върне максиманлната лексикографска разлика по модул между два произволни символа.
Пример: „zbcda“ -> 25.  Максималната лексикографка разлика по модул е между ‚a‘ и ‚z‘, или ‚z’ и ‚а‘, и тя е 25.
